# Dig Nimbella

A [dig](https://en.wikipedia.org/wiki/Dig_(command)) clone for [Nimbella Commander](https://nimbella.com/product/commander).

## Install

```
/nc csm_install dig
```

## Usage

```
/nc dig <hostname> [<type>]
```

Supported record types:

- `A` (default)
- `AAAA`
- `TXT`
- `MX`
- `NS`
- `SOA`
